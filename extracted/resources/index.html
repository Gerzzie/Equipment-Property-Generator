Custom Bonuses

Define rules for bonuses that aren't in the built-in table. Use {arg1}, {arg2}, … as placeholders. Example: name bIncreasePogi, args 1, lua IncreasePogi({arg1}), description Increase Pogi {arg1}%.

Edit Tooltip Labels

Edit the labels that appear in the in-game equipment properties tooltip. These live in EquipmentPropertiesOrder.lub next to your source .lub. Save to write changes back, then restart the client to see them.
